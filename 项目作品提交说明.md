## 项目名称

智能进化 Data Agent：基于 MinerU + DeepSeek 的财报结构化与防幻觉系统

## 提交内容说明

本项目提交一个面向“智能进化·Agent 能力评测赛道”的 Data Agent 系统，包含：

- Data Agent 系统代码与关键模块代码。
- 系统部署与运行说明。
- API 接口说明。
- 完整技术报告。
- 系统运行日志与测试结果。
- 加分材料 PPT 提纲。
