# 演示 PPT 提纲

## 第 1 页：项目标题

- 智能进化 Data Agent：基于 MinerU + DeepSeek 的财报结构化与防幻觉系统
- 赛道：智能进化·Agent 能力评测赛道
- 团队信息：提交前补充

## 第 2 页：问题背景

- 文档数据处理面临复杂版式、跨页信息、低质量 OCR 和数字幻觉问题。
- 财务报表场景中，高密度数字抽取对准确性和可追溯性要求极高。
- 单一模型直接抽取容易出现单位误读、附注编号误抽、合计关系不自洽等问题。

## 第 3 页：系统目标

- 使用 MinerU 将复杂 PDF/文档解析为结构化 Markdown。
- 使用 Data Agent 自动执行任务拆解、场景路由、指标抽取和结果校验。
- 输出可验证、可追溯、可通过 API 调用的结构化结果。

## 第 4 页：系统架构

- 接入层：FastAPI。
- 解析层：MinerU SaaS。
- 智能处理层：规则抽取 + DeepSeek 反思修正。
- 审计层：财务恒等式校验、字段缺失检查。
- 状态层：`task_db.json` 持久化任务状态。

## 第 5 页：核心创新

- MinerU 表格结构解析与 Agent 规则校验结合。
- 支持 `page_ranges`，降低大文档处理成本和等待时间。
- 财务场景引入资产负债表恒等式校验。
- 对附注编号、流动/非流动小计、归母权益等财报常见误抽进行规则修正。
- 失败不造假，无法自洽时返回明确错误。

## 第 6 页：API 工作流

1. 调用 `/v1/agent/process` 提交文档 URL 与目标指标。
2. 系统创建 `job_id` 并进入后台任务。
3. 调用 MinerU 解析文档。
4. Agent 下载 Markdown 并抽取结构化指标。
5. 校验通过后写入任务状态。
6. 调用 `/v1/agent/task_status/{job_id}` 查询结果。
